Requisitos:

* Graphviz para la generación de gráficos
	http://www.graphviz.org/

* Easy-Install para simplificar la instalación de dependencias
	http://pypi.python.org/pypi/setuptools

* Pyparsing para pydot
	http://pyparsing.wikispaces.com/
	Nota: el instalador no funciona actualmente en windows XP,
	hay que bajar el .zip y ejecutar
	C:\Python26\Scripts\easy_install.exe pyparser.zip


* Pygments para el syntax highlighting
	C:\Python26\Scripts\easy_install.exe pygments

* Pydot para la creación de ficheros .dot
	C:\Python26\Scripts\easy_install.exe pydot
